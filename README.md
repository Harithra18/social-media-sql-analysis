# Social Media Usage Analysis — SQL Project

## Overview
Analysed social media usage patterns across platforms to identify 
user engagement trends, age-group behaviour, and content preferences 
using advanced SQL queries.

## Tools & Technologies
- MySQL
- SQL — Joins, Window Functions (RANK), Subqueries, CASE, Aggregations
- Relational Database Design — Primary & Foreign Keys

## Key Analysis Performed
- Analysed user engagement across multiple social media platforms
- Ranked platforms based on user activity using SQL Window Functions
- Identified platforms with above-average usage time
- Segmented users by age group and usage behaviour
- Analysed gender-wise platform engagement patterns

## Project Files
- `Social_media.sql` — All SQL queries and analysis scripts

## Dataset
Self-created relational database with parent-child table relationships

Data Analysis · SQL · Database Design · Business Intelligence · Reporting
